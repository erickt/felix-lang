Current Development Version

Felix is free to use. It is in constant development, so the best way to use it is through Subersion: 


svn co https://felix.svn.sourceforge.net/svnroot/felix/felix/trunk felix

Older Versions

I highly recommend you use the development version. It has many bug fixes and new and interseting features. However, if you want an older version, you can find them here:

  • 1.1.3_rc1
  • 1.1.2_rc4

Prerequisites

  • Install O'Caml, 3.09 or above.
  • Install Python.
  • Install a C++ compiler. GCC and MSVC are supported.

How to Build

  • Default Options (unix-like):
    • Run autogen.sh.
    or run
    • Run configure [config options].
    • Run mk [build options].
  • Default Options (msvc):
    • Run boot.bat to initialize the system.
    • Run either win32config.bat [config options], win64config.bat [config options], win3264config.bat [config options], vista32config.bat [config options], or vista64config.bat [config options].
    • Run mk --phase=build [build options].
    • Run mk --phase=host [build options].
    • Run mk --phase=target [build options].

See configure --help and mk --help for a description of the config and build options.

Troubleshooting

For any help or suggestions with the build process, or any questions about felix itsel, try our discussion board.
